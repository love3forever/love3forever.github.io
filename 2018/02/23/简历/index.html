项目经历

浙江贵仁信息科技股份有限公司武汉分公司(2017年4月-至今)

府澴河三维实时水雨情项目

此项目主要是利用湖北省水利厅提供的实时水雨情数据，在由osgearth搭建的三维球上进行府澴河流域的信息展示。
并且将公司的水文、水动力模型计算结果展示到目标区域，进行洪水演进模拟。

在此项目中我的工作内容如下：

  1. 项目初期，在甲方未能提供实测数据的时候，使用爬虫技术获取水雨情数据，提供restful数据接口供项目使用，避免项目开发进度滞后；
  2. 根据国家标准，构建数据模型，使用SQLAlchemy操作数据库，在项目后期水利厅生产数据对接；
  3. 分析公司水文、水动力模型相关计算结果，开发数据转换工具，将计算结果转换能够直接在三维球上模拟的数据形式并提供对应数据接口；
  4. 利用水文、水动力模型数据接口，完成洪水演进在三维地球上的模拟；
  5. 基于vue+flask，开发一个用以监控后端服务状况和数据库状况的小型运维系统，为团队开发和项目实施人员提供方便；
  6. 完成项目代码部署、功能测试和项目验收相关事宜。

主要用到的技术：

  • 开发语言：python、javascript、c++
  • 数据库：mongodb、oracle、mysql、redis
  • 相关库：requests、bs4、fabric、celery、pymongo、flask、vue

武汉大学 （ 2014年9月 ~ 2016年6月）

海量瓦片数据分布式缓存系统

此项目旨在提高一个三维可视化软件的后端海量瓦片数据调度能力、简化后端数据集选择流程。
在此项目中我的工作内容：

  1. 系统技术选型
  2. 总体技术路线
  3. 系统代码编写
  4. 系统部署，功能、性能测试

在此项目中，用到的技术主要包括以下这些：

  • 数据库： mongodb、redis
  • web服务器：nginx
  • 开发语言：lua、php、C#
  • 工具：webbench、goaccess、wireshark

这个系统完成了预期目标：

  1. 有效减少了后台数据检索时间消耗：从之前5ms/条缩短到0.5ms/条；
  2. 通过nginx+lua进行的路由控制，有效减少了无效请求，避免无效数据检索；
  3. 提升了数据请求网络资源利用，带宽利用率从5mb/s提升到20mb/s（局域网环境下）；
  4. 提供给用户的缓存集群管理用具有效降低了操作复杂度，取得了不错的反馈。

开源项目

仿写网易云音乐

这个项目完成了网易云音乐部分界面的仿写，在线示例：http://service.eclipsesv.com/。
前端部分使用vue全家桶实现；前后端通过ajax进行数据通讯；使用flask构建服务：解析前端请求，并实时获取来自网易云音乐真实数据；最终完成了歌单、单曲、歌手等页面的还原。

网易云音乐数据挖掘

这个项目重新封装了仿写网易云音乐的数据接口；获取并格式化数据，在mongodb中存储；最终利用neo4j分析包括用户、单曲、专辑和艺人等数据之间的关系。

DouyuFan

这个项目主要是爬取直播网站斗鱼直播分类数据和弹幕聊天内容，并对它们进行了处理，最终通过网页图表的形式体现。
创建爬虫获取斗鱼直播分类页面，与斗鱼弹幕服务器建立socket连接来获取实时弹幕；离线弹幕数据完成数据分析并通过echarts展示分析结果。

LearningFlask

这个项目主要是用以纪录我的flask学习过程，另外其中也包含了一些包括豆瓣、知乎等网站数据的爬取。
